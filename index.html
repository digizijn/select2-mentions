@-mentions support for select2

Select2 is an awesome jQuery replacement for select boxes.

select2-mentions adds @-mentions support on top of a select2 widget by associating it with another input or textarea. Try out the example below to see how it works

Example

Simple example

Type into the textarea above, and then type @red and press enter.

Source

var selector = '#example1 .demo input.select2';
$(selector).select2({
    tags: ["red", "green", "blue", "yellow", "green", "magenta"]
});
$(selector).mentions({
    input: $('#example1 .demo textarea.mentions'),
    triggerKey: /@/,
    stopKey: /^[a-z0-9]+$/i
});

Browser support

select2-mentions is tested in Firefox, Google Chrome, and Internet Explorer 7+
